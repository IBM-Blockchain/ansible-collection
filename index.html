This ansible collection is deprecated and will no longer be updated or maintained by IBM. Please use the open source ansible collection, maintained by hyperledger-labs, which can be found here: Hyperledger Fabric Ansible Collection

IBM Blockchain collection for Ansible

This Ansible collection, enables you to automate the building of Hyperledger Fabric networks.

It supports creating these networks within - the IBM Hyperledger Fabric Support Offering - Fabric Open Source Stack (Hyperledger Labs Fabric Operations and Fabric Operations Console)

Roles are provided to install the operations console and operator; additional roles and modules can then create the Fabric Network.
